# Dashboard

**Feature name:** Dashboard  
**Route:** `/dashboard`

## What this feature does
Shows a quick summary of key business activity, such as employees, stock, tasks, and attendance.

## When users should use it
Use it at the start of the day for a quick status check.

## Step-by-step instructions
1. Open `/dashboard`.
2. Review top summary cards.
3. Check activity sections for recent updates.
4. Use refresh actions if data looks outdated.

## Expected result
You can quickly understand current status and decide what to do next.
