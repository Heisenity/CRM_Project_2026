# Login and Access

**Feature name:** Login and Access  
**Routes:** `/`, `/landing`, `/admin-login`

## What this feature does
Lets customers, staff, and admins sign in and reach the right workspace.

## When users should use it
Use this page whenever you need to log in or switch account type.

## Step-by-step instructions
1. Open `/`.
2. Choose the correct login card (Customer, Staff, or Admin).
3. Enter your ID and required details.
4. Select the login button.
5. Wait for redirect to your workspace.

## Expected result
You are signed in and moved to the correct page (for example Dashboard or Staff Portal).

## Common mistakes or errors
- Wrong ID or password: Re-enter details carefully.
- Using the wrong login card: Switch to the correct user type.
- No redirect after login: Refresh once and sign in again.
