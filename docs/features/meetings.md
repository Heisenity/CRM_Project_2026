# Meetings

**Feature name:** Meetings  
**Route:** `/meetings`

## What this feature does
Schedules and manages meetings with status, priority, and attendee details.

## When users should use it
Use it for planning upcoming meetings and tracking meeting outcomes.

## Step-by-step instructions
1. Open `/meetings`.
2. Select **New Meeting**.
3. Enter meeting details and save.
4. Use tabs such as Overview, Today, Upcoming, and Detailed View.
5. Use search and filters to find specific meetings.
6. Edit or delete meetings when needed.

## Expected result
Meetings are organized, searchable, and easy to maintain.
