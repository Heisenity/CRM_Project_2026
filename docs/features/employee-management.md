# Employee Management

**Feature name:** Employee Management  
**Routes:** `/employee-management`, `/employees`

## What this feature does
Manages employee and administrator records, including view, edit, and delete actions.

## When users should use it
Use it for staff records, role checks, and account updates.

## Step-by-step instructions
1. Open `/employee-management`.
2. Use search and role filters to find a person.
3. Open row actions to view or edit details.
4. Save changes.
5. Use delete only when the record should be removed.

## Expected result
Employee data stays current and searchable.
