# Employee Management

**Feature name:** Employee Management  
**Routes:** `/employee-management`, `/employees`

## What this feature does
Employee Management is the main page for maintaining staff records.

You can use this page to:
- View all employee accounts.
- View and manage administrator accounts.
- Search people by name, employee ID, or email.
- Filter employees by role.
- Open details, edit profile information, or remove records.
- Manage employee ID prefixes.

## When users should use it
Use this page when you need to:
- Check employee information quickly.
- Update staff details (phone, role, team, etc.).
- Review administrator accounts.
- Add a new employee from the top action button.

## Step-by-step instructions
1. Open the app and go to **Employee Management** from the sidebar.
2. On the top section, review summary cards (for example, total employees).
3. Use the search box to find a person by name, ID, or email.
4. Use the role filter to narrow the employee list.
5. Open the required tab:
   - **Employees**: for regular employee records.
   - **Admins**: for administrator account records.
   - **Prefixes**: for employee ID prefix management.
6. In the row actions menu, choose what you need:
   - **View** to read full profile details.
   - **Edit** to update information.
   - **Delete** to remove a record permanently.
7. If editing, update fields and save changes.
8. If adding a new employee, click **Add Employee** and complete the employee creation form.

## Expected result
Employee and admin records are accurate, searchable, and easy to manage.

Typical outcomes:
- You quickly find a person and verify their details.
- Profile changes are saved and visible in the list.
- Role/team updates are reflected correctly.

## Common mistakes or errors
- No employees shown: Clear filters and recheck search text.
- Wrong person edited: Confirm employee name and ID before saving.
- Save failed: Check required fields and valid formats (email, phone, etc.).
- Delete was accidental: Use delete carefully because this action may be permanent.
- Changes not visible: Refresh the page and verify again.
