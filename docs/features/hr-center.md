# HR Center

**Feature name:** HR Center  
**Route:** `/hr-center`

## What this feature does
Provides HR actions like leave management, document handling, and account support tools.

## When users should use it
Use it for day-to-day HR operations and employee support tasks.

## Step-by-step instructions
1. Open `/hr-center`.
2. Choose the needed section (leave, documents, or related actions).
3. Complete the required form/action.
4. Save or submit.
5. Use the Payslip Management button when needed.

## Expected result
HR requests and records are updated from one central page.
