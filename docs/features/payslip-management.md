# Payslip Management

**Feature name:** Payslip Management  
**Route:** `/payslip`

## What this feature does
Creates, updates, and reviews employee payslip records.

## When users should use it
Use it during payroll processing or payslip corrections.

## Step-by-step instructions
1. Open `/payslip`.
2. Select employee, month, and year.
3. Enter earnings, deductions, and reimbursement values.
4. Save or update the payslip.
5. Review saved payslip entries in the list.

## Expected result
Payslip records are generated correctly and stored for reference.
