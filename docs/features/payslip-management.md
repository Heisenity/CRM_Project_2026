# Payslip Management

**Feature name:** Payslip Management  
**Route:** `/payslip`

## What this feature does
Payslip Management helps you create and maintain monthly salary slips for employees.

You can use this page to:
- Create a new payslip for an employee.
- Update an existing payslip.
- Enter earnings, deductions, and reimbursements.
- Review previously generated payslip records.

## When users should use it
Use this page when you need to:
- Process monthly salary details.
- Correct a payslip after changes.
- Add reimbursement or deduction updates.
- Check old payslip records for reference.

## Step-by-step instructions
1. Open the app and go to **HR Center**. There, you can see at the top right **Payslip Management**.
2. Select the employee.
3. Select month and year.
4. Enter all required values:
   - Earnings (basic and other salary components)
   - Deductions (PF, ESI, tax, etc.)
   - Reimbursements (if applicable)
5. Review all amounts carefully before saving.
6. Click **Generate** or **Update** (based on whether it is a new or existing record).
7. Check the payslip list below to confirm the record appears correctly.
8. If needed, open an existing record and update it.

## Expected result
Payslip records are saved correctly and available for later review or correction.

Typical outcomes:
- Monthly payslip is generated for the selected employee.
- Net pay reflects entered earnings and deductions.
- Updated payslip values are visible in the records list.


## Common mistakes or errors
- Payslip not saved: Check required fields (employee, month, year).
- Wrong totals: Recheck numeric values and decimal points.
- Duplicate monthly record: Verify month/year before creating a new payslip.
- Update not visible: Refresh and reopen the payslip record.
- Invalid number input: Enter only valid numeric values in amount fields.
