# Field Engineer Attendance

**Feature name:** Field Engineer Attendance  
**Routes:** `/field-engineer-attendance`, `/attendance-management`

## What this feature does
Field Engineer Attendance helps you monitor and manage attendance for field staff in one place.

You can use this page to:
- View daily attendance records for field engineers.
- Search staff quickly by name or employee details.
- Filter records to find specific attendance status.
- Check clock-in and clock-out times.
- Correct attendance times when needed.

## When users should use it
Use this page when you need to:
- Confirm who has reported for field work today.
- Review late, missing, or incomplete attendance entries.
- Correct incorrect clock-in or clock-out times.
- Check attendance history for reporting.

## Step-by-step instructions
1. Open the app, open the left sidebar, and click **Field Engineer**.
2. Use the search box to find a specific field engineer.
3. Apply filters (status or other available filters) to narrow the list.
4. Review each row:
   - Employee details
   - Clock-in time
   - Clock-out time
   - Attendance status
5. If a time is incorrect, open row actions and click **Edit Attendance**.
6. Update the clock-in/clock-out values and save.
7. Confirm the updated values appear correctly in the table.
8. Use export or refresh actions (if available) for reporting or latest data.

## Expected result
Field attendance records are accurate, current, and easy to review.

Typical outcomes:
- You quickly see who is present or missing.
- Incorrect time entries are corrected.
- Attendance data is ready for daily operations and reporting.

## Common mistakes or errors
- Employee not visible: Clear filters and search again.
- No records shown: Check the selected date or refresh the page.
- Edit not saved: Ensure time fields are valid before saving.
- Wrong record edited: Verify employee name and ID before submitting changes.
- Status looks incorrect: Refresh once and confirm the latest entry.
