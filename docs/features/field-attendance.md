# Field Engineer Attendance

**Feature name:** Field Engineer Attendance  
**Routes:** `/field-engineer-attendance`, `/attendance-management`

## What this feature does
Tracks attendance for field engineers with search, filters, and edit options.

## When users should use it
Use it to monitor field attendance and fix incorrect time entries.

## Step-by-step instructions
1. Open `/field-engineer-attendance`.
2. Search for an employee or apply filters.
3. Review clock-in, clock-out, and status.
4. Use edit action if attendance times need correction.
5. Save changes.

## Expected result
Field attendance records stay accurate and up to date.
