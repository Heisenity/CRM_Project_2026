# Task Management

**Feature name:** Task Management  
**Routes:** `/task-management`, `/attendance`

## What this feature does
Helps teams assign tasks, track live task status, and review task history.

## When users should use it
Use it to assign work, monitor progress, or check completed tasks.

## Step-by-step instructions
1. Open `/task-management`.
2. Use search and filters to find employees or tasks.
3. Select **Assign** to create a new task.
4. Update task status when work starts or ends.
5. Use the History tab to review past task updates.

## Expected result
Tasks are assigned clearly, status stays up to date, and history is available for review.

## Common mistakes or errors
- Task not visible: Check active filters.
- Cannot assign task: Required fields may be missing.
- Wrong status shown: Refresh and verify last update.
