# Task Management

**Feature name:** Task Management  
**Routes:** `/task-management`, `/attendance`

## What this feature does
Task Management is used to assign work, monitor task progress, and track status changes over time.

You can use this page to:
- Assign tasks to employees.
- Edit existing task details.
- Unassign/cancel tasks when plans change.
- Track current task progress in real time.
- Review complete task history for reporting.
- Search and filter records quickly.

## When users should use it
Use this page when you need to:
- Give new tasks to field or office staff.
- Check who is currently working on what.
- Follow pending, in-progress, and completed tasks.
- Review past task activity for audits or follow-up.

## Step-by-step instructions
1. Open the app, then go to **Task Management** in the left sidebar.
2. On the main page, review the top summary cards (total, pending, in progress, completed, no task).
3. Use search and filters to find specific employees or tasks.
4. To create a new task:
   - Click **Assign**.
   - Fill task details (employee, task title, category, location, and related info if needed).
   - Save the task.
5. To update a task:
   - Open row actions for that employee/task.
   - Click **Edit Task**.
   - Update details and save.
6. To remove a task assignment:
   - Open row actions.
   - Click **Unassign Task**.
   - Confirm the action.
7. Use the **Live** tab to monitor current task status.
8. Use the **History** tab to review past status changes and assignments.
9. Export data (if available) when you need reports.

## Expected result
Tasks are assigned correctly, progress is visible in real time, and all status changes are traceable.

Typical outcomes:
- Team members receive clear task assignments.
- Managers can quickly identify delayed or pending work.
- History records provide reliable tracking for review.

## Common mistakes or errors
- Task not visible: Check active search text and filters.
- Employee not found during assignment: Confirm employee is active and selected correctly.
- Cannot assign task: Complete required fields before saving.
- Wrong status shown: Refresh the page and verify the latest update.
- Task unassigned by mistake: Reassign the task from the Assign flow.
- History appears empty: Adjust filters/date and check again.
