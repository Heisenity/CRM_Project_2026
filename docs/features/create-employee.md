# Add New Employee

**Feature name:** Add New Employee  
**Route:** `/employees/new`

## What this feature does
Creates a new employee account with profile and work details.

## When users should use it
Use it when onboarding a new employee.

## Step-by-step instructions
1. Open `/employees/new`.
2. Enter role, employee ID, and personal details.
3. Add optional profile photo and additional information.
4. Select **Create Employee**.
5. Confirm the success message.

## Expected result
A new employee account is created and ready for login/use.
