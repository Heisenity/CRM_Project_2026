# Add New Employee

**Feature name:** Add New Employee  
**Route:** `/employees/new`

## What this feature does
Add New Employee helps you create a complete employee account for a new staff member.

You can use this page to:
- Create login-ready employee records.
- Set role and identity details.
- Add optional profile and payroll-related information.
- Verify entered details before final submission.

## When users should use it
Use this page when you need to:
- Onboard a new employee.
- Add a field engineer or office employee account.
- Create a staff profile that can later be managed in Employee Management.

## Step-by-step instructions
1. Open the app and go to **Employee Management**.
2. Click **Add Employee** (top-right action).
3. Select the employee role (for example Field Engineer or Office Employee).
4. Enter employee ID details:
   - Use generated ID, or
   - Enter a custom ID if your process requires it.
5. Enter basic profile details:
   - Full name
   - Email (if available)
   - Password
   - Phone number
6. Add work details:
   - Designation
   - Team (if applicable)
   - Leave balances
   - Salary details (if required)
7. Add personal/compliance details if needed:
   - Address
   - Aadhar/PAN
   - UAN/ESI
   - Bank account details
8. Upload a profile photo (optional, but recommended).
9. Review the **Employee Preview** panel to confirm details.
10. Click **Create Employee**.
11. Confirm the success message appears.

## Expected result
A new employee account is created successfully and is ready for login/use based on assigned role and access.

Typical outcomes:
- Employee appears in Employee Management list.
- Login credentials are set for the new employee.
- Basic HR and profile records are available for future updates.

## Common mistakes or errors
- Employee creation fails: Check all required fields are filled.
- Invalid employee ID format: Use allowed format and retry.
- Duplicate employee ID: Enter a unique ID.
- Invalid phone/email/number fields: Correct formatting and submit again.
- Photo upload fails: Retry with a smaller, supported image file.
- New employee not visible: Refresh Employee Management and search by employee ID.
