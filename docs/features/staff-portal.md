# Staff Portal

**Feature name:** Staff Portal  
**Route:** `/staff-portal`

## What this feature does
Acts as the main home page for staff, with quick access to allowed tools.

## When users should use it
Use it after login to open daily work features from one place.

## Step-by-step instructions
1. Open `/staff-portal`.
2. Review available feature cards or menu items.
3. Select the feature you need (Attendance, Tickets, Projects, etc.).
4. Return to the portal to switch to another feature.

## Expected result
You can reach all features assigned to your account from one page.
