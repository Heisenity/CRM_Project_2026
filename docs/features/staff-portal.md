# Staff Portal

**Feature name:** Staff Portal  
**Route:** `/staff-portal`

## What this feature does
Staff Portal is the main workspace for staff users after login.

You can use this page to:
- Access daily tools from one place.
- Open assigned features quickly.
- View work-related shortcuts and cards.
- Move between modules without searching menus repeatedly.

## When users should use it
Use this page when you need to:
- Start your daily work after login.
- Open attendance, tickets, projects, tasks, or other assigned tools.
- Check notifications or pending actions.
- Switch between multiple staff features during the day.

## Step-by-step instructions
1. Open the app and sign in with a staff account.
2. After login, the system opens **Staff Portal**.
3. Use the left sidebar (or menu) to view available feature options.
4. Review dashboard cards/shortcuts shown on the page.
5. Click the feature you need (for example Attendance, Tickets, Projects, Stock, HR, or Team-related pages).
6. Complete your work in that feature page.
7. Return to **Staff Portal** to open another feature.
8. If a needed feature is missing, contact admin or update access in **Staff Feature Access**.

## Expected result
You can access all permitted staff features from one central page and switch between tasks quickly.

Typical outcomes:
- Staff users see only features assigned to them.
- Daily navigation becomes faster with one entry point.
- Pending items are easier to notice and act on.

## Common mistakes or errors
- A feature is not visible: Your account may not have permission for that feature.
- Feature opens and redirects back: Access may be restricted for your role.
- Sidebar not visible on mobile: Open the menu button to show sidebar options.
- Page content looks outdated: Refresh the page and sign in again if needed.
- Wrong account view: Sign out and log in with the correct staff account.
