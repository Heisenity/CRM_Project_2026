# Team Management

**Feature name:** Team Management  
**Route:** `/teams`

## What this feature does
Lets users create, update, and remove teams.

## When users should use it
Use it when organizing employees into working teams.

## Step-by-step instructions
1. Open `/teams`.
2. Select **Create Team**.
3. Enter team details and save.
4. Use edit action to update team information.
5. Use delete action only for teams no longer needed.

## Expected result
Teams are organized and maintained correctly.
