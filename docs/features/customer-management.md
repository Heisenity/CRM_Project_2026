# Customer Management

**Feature name:** Customer Management  
**Route:** `/customers`

## What this feature does
Stores customer records and helps you create, update, view, and manage customer details.

## When users should use it
Use it when onboarding new customers or updating existing customer information.

## Step-by-step instructions
1. Open `/customers`.
2. Select **Add Customer**.
3. Enter customer details and create the record.
4. Use search to find an existing customer.
5. Use view, edit, or delete actions as needed.

## Expected result
Customer records remain organized and up to date.
