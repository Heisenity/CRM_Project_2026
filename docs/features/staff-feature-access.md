# Staff Feature Access

**Feature name:** Staff Feature Access  
**Route:** `/staff-feature-access`

## What this feature does
Controls which features each staff member can open.

## When users should use it
Use it when enabling or removing feature access for staff.

## Step-by-step instructions
1. Open `/staff-feature-access`.
2. Find and select a staff member.
3. Turn feature permissions on or off.
4. Save access changes.
5. Ask the staff user to refresh their portal.

## Expected result
Staff only see and use features assigned to them.

- Changes not visible to staff: Ask them to sign out and sign in again.
- Wrong user updated: Confirm employee name/ID before saving.
