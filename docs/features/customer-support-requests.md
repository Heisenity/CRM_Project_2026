# Customer Support Requests

**Feature name:** Customer Support Requests  
**Route:** `/customer-support-requests`

## What this feature does
Shows incoming customer support requests and helps staff accept requests and create tickets.

## When users should use it
Use it when handling pending customer requests.

## Step-by-step instructions
1. Open `/customer-support-requests`.
2. Review **Pending Requests**.
3. Select **Accept** on a request.
4. Open the accepted request and choose **Create Ticket**.
5. Fill ticket fields and save.

## Expected result
Each support request is accepted by staff and converted into a trackable ticket.

## Common mistakes or errors
- Cannot create ticket: Accept the request first.
- Missing problem type or priority: Fill required fields before submit.
