# Vehicle Management

**Feature name:** Vehicle Management  
**Route:** `/vehicles`

## What this feature does
Manages vehicles and related fuel bill workflows.

## When users should use it
Use it for adding vehicles, assigning status actions, and reviewing fuel bill records.

## Step-by-step instructions
1. Open `/vehicles`.
2. Select **Add New Vehicle** for a new entry.
3. Fill vehicle details and save.
4. Use search/filter to find existing vehicles.
5. Review fuel bill entries and approve/reject when required.

## Expected result
Vehicle records and bill actions stay organized and current.
