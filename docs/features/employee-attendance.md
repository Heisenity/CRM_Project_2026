# Employee Attendance

**Feature name:** Employee Attendance  
**Route:** `/employee-attendance`

## What this feature does
Lets employees mark daily clock-in/clock-out and follow attendance-related actions.

## When users should use it
Use it at the start and end of each working day.

## Step-by-step instructions
1. Open `/employee-attendance`.
2. Go to the **Daily Clock-in/out** section.
3. Select clock-in when starting work.
4. Select clock-out when ending work.
5. Check current attendance status.

## Expected result
Your day attendance is recorded and visible for approval/review.
