# Office Attendance

**Feature name:** Office Attendance  
**Route:** `/inoffice-attendance`

## What this feature does
Office Attendance helps you monitor and manage attendance records for office staff.

You can use this page to:
- View daily attendance records for office employees.
- Search employees quickly.
- Filter records by attendance status.
- Check clock-in and clock-out times.
- Correct incorrect time entries.

## When users should use it
Use this page when you need to:
- Check who is present, late, or absent today.
- Review attendance before payroll or reporting.
- Fix wrong clock-in or clock-out entries.
- Confirm attendance updates were saved correctly.

## Step-by-step instructions
1. Open the app and go to **Office Attendance** from the sidebar.
2. Use the search box to find a specific employee.
3. Apply status filters (for example, Present, Late, or Absent) if needed.
4. Review each attendance row:
   - Employee details
   - Clock-in time
   - Clock-out time
   - Attendance status
5. If an entry is incorrect, open row actions and choose **Edit Attendance**.
6. Update the clock-in/clock-out values and save.
7. Confirm the updated values appear correctly in the table.
8. Use refresh or export options (if available) for latest data or reporting.

## Expected result
Office attendance records are accurate, easy to search, and ready for daily use.

Typical outcomes:
- You quickly identify present/late/absent employees.
- Incorrect attendance times are corrected.
- Updated attendance records are visible immediately.

## Common mistakes or errors
- No records visible: Clear filters and refresh the page.
- Employee not found: Check spelling and try searching by employee ID.
- Wrong status shown: Recheck clock-in/clock-out values and save again.
- Edit not saved: Ensure time fields are valid before submitting.
- Changes not visible: Refresh once and wait a few seconds.
