# Office Attendance

**Feature name:** Office Attendance  
**Route:** `/inoffice-attendance`

## What this feature does
Manages daily attendance for office employees.

## When users should use it
Use it for attendance checks, corrections, and status review.

## Step-by-step instructions
1. Open `/inoffice-attendance`.
2. Use search and status filter options.
3. Review clock-in/clock-out rows.
4. Edit attendance time where needed.
5. Save and confirm update.

## Expected result
Office attendance is clear, searchable, and corrected when needed.
