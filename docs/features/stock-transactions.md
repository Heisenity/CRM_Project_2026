# Stock Transactions

**Feature name:** Stock Transactions  
**Route:** `/stock`

## What this feature does
Stock Transactions helps you record and review inventory movement.

You can use this page to:
- Track stock going in and out.
- View complete transaction history.
- Search transactions by product, barcode, or user details.
- Filter records to find specific entries quickly.
- Export stock transaction data for reports.

## When users should use it
Use this page when you need to:
- Check current stock movement.
- Verify who handled a stock transaction.
- Review inventory activity for a date or period.
- Prepare stock reports for management or audit.

## Step-by-step instructions
1. Open the app, then go to **Stock Management** in the left sidebar.
2. Review the transaction list on the page.
3. Use search to find entries by product name, barcode, or staff details.
4. Apply filters (if available) to narrow records by type/status/date.
5. To add a new movement entry, click the add/new transaction action.
6. Enter transaction details carefully (product, quantity, type, and remarks if needed).
7. Save the transaction.
8. Confirm the new entry appears in the transaction list.
9. Use **Export** when you need a downloadable report.

## Expected result
Stock movement records are accurate, searchable, and ready for review.

Typical outcomes:
- Every stock movement is logged with details.
- You can quickly find past transactions.
- Reports can be generated when required.

## Common mistakes or errors
- Transaction not visible: Clear filters and refresh the page.
- Wrong item selected: Verify product name/SKU/barcode before saving.
- Quantity entered incorrectly: Recheck numeric values before submit.
- Save failed: Ensure required fields are completed.
- Export missing data: Confirm filters/date range before exporting.
