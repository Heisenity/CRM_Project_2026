# Stock Transactions

**Feature name:** Stock Transactions  
**Route:** `/stock`

## What this feature does
Tracks inventory movement and stock-related transactions.

## When users should use it
Use it when checking stock movement, reviewing history, or exporting records.

## Step-by-step instructions
1. Open `/stock`.
2. Review inventory transaction list.
3. Use search and filters to narrow results.
4. Add new stock transaction when needed.
5. Export data if a report is required.

## Expected result
Stock movement is traceable and easy to audit.
