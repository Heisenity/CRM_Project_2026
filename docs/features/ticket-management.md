# Support Ticket Management

**Feature name:** Support Ticket Management  
**Routes:** `/tickets`, `/tickets/new`

## What this feature does
Creates, tracks, updates, and closes support tickets.

## When users should use it
Use it whenever an issue is reported and needs follow-up.

## Step-by-step instructions
1. Open `/tickets` to view all tickets.
2. Use tabs and filters (All/Open/Resolved/Closed).
3. Select **Create Ticket** or open `/tickets/new`.
4. Fill ticket details and submit.
5. Update ticket status as work progresses.

## Expected result
Issues are tracked from creation to closure with clear status history.

- Ticket not showing: Check filter tab and search text.
- Cannot submit ticket: Fill required fields like problem type and priority.
