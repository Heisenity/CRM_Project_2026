# Tender Management

**Feature name:** Tender Management  
**Route:** `/tenders`

## What this feature does
Tender Management helps you handle the full tender process from creation to final tracking.

You can use this page to:
- Create and maintain tender records.
- Track tender status (draft, submitted, approved, rejected, etc.).
- Upload and manage tender documents.
- Record and monitor EMD (Earnest Money Deposit) details.
- Search and filter tenders quickly.

## When users should use it
Use this page when you need to:
- Start a new tender process.
- Update tender details or status.
- Attach required tender documents.
- Track EMD payments and updates.
- Review tender data for follow-up or reporting.

## Step-by-step instructions
1. Open the app, then go to **Tender Management** in the left sidebar.
2. Review the tender list and current status summary.
3. Click **Create New Tender**.
4. Enter tender details (name, department, status, required documents, and related values).
5. Save the tender.
6. Use search and filters to find a specific tender quickly.
7. Open tender details to manage additional sections.
8. In the **Documents** tab, upload required files and review uploaded records.
9. In the **EMD** tab, add EMD entries and update their current status.
10. Edit tender details whenever there are process changes.

## Expected result
Tender records stay complete, searchable, and updated with required documents and EMD history.

Typical outcomes:
- New tenders are added with clear status.
- Required documents are linked to the correct tender.
- EMD records are tracked correctly through each stage.

## Common mistakes or errors
- Tender not saved: Fill all required fields before submitting.
- Tender not found: Clear filters and search again.
- Missing documents: Upload required files in the Documents tab.
- EMD status incorrect: Edit the EMD record and save updated values.
- Duplicate tender entry: Search by tender name/number before creating a new one.
