# Tender Management

**Feature name:** Tender Management  
**Route:** `/tenders`

## What this feature does
Manages tender records, required documents, and EMD tracking.

## When users should use it
Use it when creating tenders and tracking submission progress.

## Step-by-step instructions
1. Open `/tenders`.
2. Select **Create New Tender**.
3. Fill tender details and save.
4. Open tender details to manage documents.
5. Add and track EMD records as needed.

## Expected result
Tender lifecycle is tracked with clear status, documents, and EMD updates.
