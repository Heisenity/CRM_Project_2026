# Project Management

**Feature name:** Project Management  
**Routes:** `/projects`, `/enterprise-projects`

## What this feature does
Lets users create, manage, and monitor project details, updates, payments, and related products.

## When users should use it
Use it for ongoing project tracking and project progress reporting.

## Step-by-step instructions
1. Open `/projects`.
2. Select **New Project** to add a project.
3. Fill required details and save.
4. Use search and filters to find projects.
5. Open a project to add updates, payments, or product details.

## Expected result
Projects are organized and progress is easy to track.

## Common mistakes or errors
- Project not found: Clear filters and search again.
- Save fails: Check required fields.
- Update not visible: Refresh project details.
