# Project Management

**Feature name:** Project Management  
**Routes:** `/projects`, `/enterprise-projects`

## What this feature does
Project Management helps you plan, track, and manage project work from start to finish.

You can use this page to:
- Create new projects.
- Track project status and priority.
- Link projects with customers.
- Add project updates.
- Add and track project payments.
- Attach products/items related to the project.

## When users should use it
Use this page when you need to:
- Start a new project.
- Monitor project progress.
- Record project-related payment activity.
- Keep project details updated for reporting.

## Step-by-step instructions
1. Open the app, then click **Project Management** in the left sidebar.
2. Click **New Project**.
3. Enter project details (name, description, dates, status, priority, and customer if applicable).
4. Save the project.
5. Use search and filters to find any project quickly.
6. Open a project record to manage details.
7. Use project tabs/actions to:
   - Add progress updates.
   - Add payment entries.
   - Add related products/items.
8. Edit project details whenever scope, timeline, or status changes.
9. Delete a project only if it is no longer required.

## Expected result
Projects are organized, easy to track, and updated with the latest progress and payment information.

Typical outcomes:
- New projects appear in the project list.
- Progress updates are visible in project details.
- Payment and product records stay linked to the correct project.

## Common mistakes or errors
- Project not visible: Clear filters and search again.
- Save failed: Check required fields before submitting.
- Wrong customer linked: Edit project details and select the correct customer.
- Update/payment not visible: Refresh the page and reopen project details.
- Duplicate project created: Search by project name before creating a new one.
